أدوات بناء منصة OpenEdx

لبناء وتطوير منصة OpenEdx والخدمات المرتبطة بها يتم إستخدام مجموعة أدوات تقنية كالتالي :

  1. نظام التشغيل Ubuntu
  2. الأجهزة الإفتراضية بواسطة Vagrant, Docker
  3. قواعد البيانات MySQL, MongoDB
  4. خادم الويب Nginx, Gunicorn
  5. لغة البرمجة وإطار العمل Python, Django
  6. الواجهات والتفاعلية بإستخدام Javascript, React
  7. للعمليات غير المتزامنة DjangoCelery
